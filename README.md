## Features

- **Interactive Map** - Leaflet.js-based geospatial visualization
- **213 Monitoring Stations** - Complete coverage across Telangana
- **Color-Coded WQI** - Green (Safe), Orange (Polluted), Red (Highly Polluted)
- **Multiple Base Maps** - Geographic, Terrain, Satellite, Street views
- **Responsive Design** - Works on desktop and mobile devices
- **Support Link** - Contribute via Buy Me a Coffee
