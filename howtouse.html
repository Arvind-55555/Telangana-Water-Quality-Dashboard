How to Use the Dashboard

Dashboard Features

1. Interactive Map

The main dashboard displays an interactive map of Telangana with all 213 water quality monitoring stations.

Navigation: Support
  • Click and drag to pan the map
  • Use mouse wheel or +/- buttons to zoom in/out
  • Double-click on any area to zoom in quickly
Support

2. Station Markers

Each circle marker represents a water quality monitoring station, color-coded by Water Quality Index (WQI):

  • Green circles: Safe water (WQI ≥ 70)
  • Orange circles: Polluted water (WQI 45-69)
  • Red circles: Highly polluted water (WQI < 45)
Interacting with Stations: Support
  • Hover: See station name and WQI value
  • Click: Open detailed popup with all water quality parameters
Support

3. Base Map Layers

Switch between different map views using the layer control in the top-right corner:

  • Geographic Map: Default view with terrain features
  • Terrain Map: Topographic view showing elevation
  • Satellite Imagery: Aerial photography view
  • Street Map: Detailed street-level view

4. Understanding Water Quality Data

Water Quality Index (WQI) is a composite score (0-100) calculated from multiple parameters:

  • 70-100: Safe/Potable - Suitable for drinking after basic treatment
  • 45-69: Polluted - Requires advanced treatment before use
  • 0-44: Highly Polluted - Not suitable for most uses without extensive treatment

5. Key Parameters Displayed

  • DO (Dissolved Oxygen): Essential for aquatic life (ideal: ≥ 5 mg/L)
  • pH: Acidity/alkalinity (ideal: 6.5-8.5)
  • BOD (Biochemical Oxygen Demand): Organic pollution indicator (ideal: ≤ 3 mg/L)
  • COD (Chemical Oxygen Demand): Chemical pollutant load (ideal: ≤ 10 mg/L)
  • Total Coliform: Bacterial contamination (ideal: ≤ 50 MPN/100ml)
  • TDS (Total Dissolved Solids): Mineral content (ideal: ≤ 500 mg/L)
